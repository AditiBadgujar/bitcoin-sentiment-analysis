# Trader Performance Analysis Based on Bitcoin Market Sentiment

## Objective
Analyze the relationship between Bitcoin market sentiment and trader performance using historical trading data and the Fear & Greed Index.

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Files Included
- Trader_Performance_Analysis.docx
- Analysis Notebook

## Key Findings
- Extreme Greed had the highest average profit per trade.
- Fear generated the highest total profit.
- Extreme Greed achieved the highest win rate.
- Fear had the highest trading activity.# bitcoin-sentiment-analysis
